# Drishti Platform - AI-Driven Government Complaint Management

This is a Django-based platform for managing citizen grievances with AI assistance.

## Features
- **Citizen Dashboard**: File and track complaints.
- **AI Categorization**: Automatically categorizes complaints using NLP.
- **Role-Based Access**: Separate dashboards for Citizens, Officers, and Admins.
- **Analytics**: Visual insights for administrators.

## Deployment
Hosted on Hugging Face Spaces using Docker.
